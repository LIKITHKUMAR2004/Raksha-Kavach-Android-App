
# 🛡️ Raksha-Kavach Android App

## 📌 Project Overview

Raksha-Kavach is an Android-based workplace safety management application developed to improve worker safety awareness and reduce workplace accidents through smart digital safety checklists and monitoring features.

The application helps workers and supervisors verify safety precautions before starting dangerous tasks such as welding, electrical work, and construction activities. It provides interactive safety checklists, incident reporting, safety score analysis, and educational quizzes to ensure proper safety compliance.

This project was developed as a final-year academic project using Java and Android Studio.

---

# 🚀 Main Features

## 🤖 AI Smart Safety Generator
- User can enter any task name
- Application intelligently detects task category
- Automatically generates suitable safety checklist
- Example tasks:
  - Welding
  - Electrical Work
  - Construction
  - Industrial Work

---

## 🔥 Welding Safety Checklist
- Helmet verification
- Heat-resistant gloves check
- Ventilation verification
- Fire extinguisher availability
- Cable damage inspection
- Steel-toed boot verification

---

## ⚡ Electrical Safety Checklist
- Power supply OFF verification
- Insulated gloves check
- Cable insulation inspection
- Grounding verification
- Emergency shutdown accessibility
- Water/moisture safety check

---

## 🏗️ Construction Safety Checklist
- Safety helmet verification
- Scaffolding inspection
- Ladder damage inspection
- Fall protection verification
- First aid availability
- Safety net installation check

---

# 📋 Interactive Checklist System
- Users can mark completed safety items
- Real-time checklist progress tracking
- Ensures all safety checks are completed before work starts
- Improves workplace discipline and awareness

---

# 🚨 Incident Reporting Module
- Workers can report incidents directly through the app
- Simple and user-friendly reporting interface
- Helps maintain safety records and track unsafe conditions

---

# 📊 Safety Score Analysis
- Calculates worker safety compliance score
- Displays performance percentage
- Provides recommendations based on score
- Score Categories:
  - Excellent
  - Good
  - Fair
  - Poor

- Interactive quiz system

- Multiple-choice safety questions
- Educational awareness feature
